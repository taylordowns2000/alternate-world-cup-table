# Alternate World Cup Table

Points, goals, and goal difference for all 48 teams in World Cup 2026, per
million people, per billions of dollars of GDP, or per GDP/capita.

## Why?

I've often found myself wondering how teams are performing based on a number of
different factors. Population size (how many people might be able to play in a
nation?), GDP (how large is that nation's economy by some inadequate but fairly
standardized measure?), or GDP/capita (how wealthy are the people in that
country?) and surprised that no basic, up-to-date table like this existed for
WC2026.
